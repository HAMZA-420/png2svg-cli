# png2svg

## Installation

```batch
npm i -g png2svg-cli
```

## How to use!

```batch
png2svg-cli [...list-of-file.png]
```

### Example

- Single file

```batch
png2svg-cli test01.png
```

- Multiple files

```batch
png2svg-cli test01.png test02.png
```

- All files

```batch
png2svg-cli *.svg
```
