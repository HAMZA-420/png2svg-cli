# png2svg-cli

## Installation

```batch
npm i -g svg2png-cli
```

## How to use!

```batch
svg2png-cli [...list-of-file.png]
```

### Example

```batch
svg2png-cli test01.png test02.png
```

All of the sources for the conversion are from https://github.com/webbestmaster/pixel-art-2-svg.
